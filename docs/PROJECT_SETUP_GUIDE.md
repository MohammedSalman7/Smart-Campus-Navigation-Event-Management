# Smart Campus Setup Guide

## Start Backend

cd server
npm install
npm run dev

Runs on:
http://localhost:5000

---

## Start Frontend

cd client
npm install
npm run dev

Runs on:
http://localhost:5173

---

## Environment Variables

PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
